# Premium Deposyt Replica

A high-fidelity, single-page merchant solutions website inspired by **Deposyt.com** and themed with the premium aesthetic of **MarketLyne.com**.

## ✨ Features
- **Modern Fintech UI**: Dark theme (`#161616`) with Gold accents (`#b49146`).
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- **Glassmorphic Components**: High-end blurred backgrounds and borders.
- **AI-Generated Assets**: Custom hero imagery and partner logos.
- **Static Hosting Friendly**: Perfect for cPanel or any static file host.

## 🛠️ Technology Stack
- **HTML5** & **Vanilla JavaScript**
- **Tailwind CSS** (via CDN for rapid deployment, or local build)
- **Google Fonts** (Outfit & Inter)

## 📁 Project Structure
- `index.html`: Main website entry point.
- `assets/img/`: Project images and icons.
- `tailwind.config.js`: Custom color and font configuration.
- `src/input.css`: Custom CSS utilities.

## 🚀 Deployment (cPanel)
1. Zip the folder contents (excluding `node_modules`).
2. Upload to your `public_html` directory via cPanel File Manager.
3. Extract and enjoy.
